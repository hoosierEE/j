Release Notes for the Image3 Addon; minor changes for J5.02a
by Zachary X. Reiter and Clifford A. Reiter
June 2003

The release J5.02a changed the technique for referencing the addon path names, so these have been updated. Also, some auxiliary scripts have been added. These include:

  • filter1.ijs which gives some filters for correcting image defects. See the examples at the end of the script.
  • color_space.ijs which gives tools for converting between color spaces. See the examples at the end of the script.
  • transform_m.ijs which gives tools for correcting barrel distortion and doing tilt and small rotation adjustments. See Transform Image help file
Release Notes for the Image3 Addon; minor changes for J5.01b
by Zachary X. Reiter and Clifford A. Reiter
November 2002

The release of J5.01b changed jaddonpath to jaddonspath and this required minor changes to most of the scripts and labs. We also:

  • Use jtemppath to obtain temporary directories for the image3 labs
  • Changed bmp read/writes so that they work properly on the Mac release
  • Localized varables and fixed buffer size in avi movie functions
  • Fixed various spelling and grammar errors
See the main release notes for the main features new to image3.
